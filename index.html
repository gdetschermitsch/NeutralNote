Quick Use Guide:

1. Click Run Check and ensure all systems are ready.
2. Select your Audio Input for bite audio + meter and Language.
3. Enter Topic Title and Speakers Present.
4. Click Start Session to begin live transcription.
Note: the selected audio input controls the bite-audio capture and input meter. Live transcription still depends on how the browser routes speech recognition.

5. Speak clearly — transcript appears in real time.
6. Click Stop Session when finished.
7. Review and edit Transcript Bites if needed.
8. Click Export Session to save your data.
9. Use Clear Session to reset everything.

Best performance in Chromium-based browsers.
